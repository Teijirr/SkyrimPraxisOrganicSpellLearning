# Praxis - Organic Spell Learning

Praxis - Organic Spell Learning is a Skyrim mod, zero-script, pure SKSE plugin that completely reworks how your character progresses as a mage. Forget buying spell tomes; true mastery comes from deliberate practice. 

Because this uses [CommonLibSSE NG](https://github.com/CharmedBaryon/CommonLibSSE-NG), it supports Skyrim SE, AE, GOG, and VR.
